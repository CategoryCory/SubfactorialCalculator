Subfactorial Calculator

A subfactorial of n is the number of permutations of n objects where no object appears in its natural place, which is also referred to as a "derangement." For example, the subfactorial of 3 is 2, because there are only two ways to arrange {1, 2, 3} in which no object is in its natural place...{2, 3, 1} and {3, 1, 2}.

The subfactorial of a number can be found using the following formula:

$$!n=n!\sum_{k=0}^{n}\frac{(-1)^k}{k!}$$
